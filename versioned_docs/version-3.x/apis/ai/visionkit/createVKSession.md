## 类型

```tsx
(version: keyof Version) => VKSession
```

## 参数

| 参数 | 类型 |
| --- | --- |
| version | `keyof Version` |

### Version

vision kit 版本

| 参数 | 说明 |
| --- | --- |
| v1 | 旧版本 |
| v2 | v2 版本，目前只有 iOS 基础库 2.22.0 以上支持 |

### Track

跟踪配置

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| plane | `Plane` | 平面跟踪配置 |

### Plane

平面跟踪配置

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| mode | `keyof PlaneMode` | 平面跟踪配置模式 |

### PlaneMode

平面跟踪配置模式合法值

| 参数 | 说明 |
| --- | --- |
| 1 | 检测横向平面 |
| 2 | 检测纵向平面，只有 v2 版本支持 |
| 3 | 检测横向和纵向平面，只有 v2 版本支持 |
