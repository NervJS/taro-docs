## 方法

| 参数 | 类型 | 只读 | 说明 |
| --- | --- | :---: | --- |
| kind | `keyof Kind` | 是 | 轨道类型 |
| duration | `number` | 是 | 轨道长度 |
| volume | `number` | 否 | 音量，音频轨道下有效，可写 |

## 参数

### Kind

| 参数 | 说明 |
| --- | --- |
| audio | 音频轨道 |
| video | 视频轨道 |
