## 方法

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| bytesRead | `number` | 实际读取的字节数 |
| arrayBuffer | `ArrayBuffer` | 被写入的缓存区的对象，即接口入参的 arrayBuffer |
