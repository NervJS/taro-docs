## 参数

### observe

#### Option

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| type | `keyof EntryType` | 指标类型。不能和 entryTypes 同时使用 |
| entryTypes | `(keyof EntryType)[]` | 指标类型列表。不能和 type 同时使用。 |

#### EntryType

| 参数 | 说明 |
| --- | --- |
| navigation | 路由 |
| render | 渲染 |
| script | 脚本 |
