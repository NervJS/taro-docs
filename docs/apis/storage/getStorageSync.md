## 类型

```tsx
<T = any>(key: string) => T
```

## 参数

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| key | `T` | 本地缓存中指定的 key |

## 示例代码

```tsx
try {
  var value = Taro.getStorageSync('key')
  if (value) {
    // Do something with return value
  }
} catch (e) {
  // Do something when catch error
}
```
